## 1.1 方案介绍

主要记录了关于六轮双驱差速机器人教研版方案的开发

基于这个方案，能够使得初学者能够快速得搭建起来整个机器人系统，从而使得学习者能够通过使用这个软件，快速地对整个机器人系统有系统的认识。
